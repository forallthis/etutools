# OKTools Chrome Extension - 测试指南

## 📦 安装步骤

### 1. 加载扩展到Chrome

1. 打开Chrome浏览器
2. 访问 `chrome://extensions/`
3. 打开右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目的 `dist` 文件夹
6. 扩展加载成功！

### 2. 验证安装

- 在Chrome工具栏应该看到OKTools图标
- 右键点击图标，应该看到"在侧边栏中打开"选项
